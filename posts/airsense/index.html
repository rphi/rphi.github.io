AirSense - personal air quality monitor

April 3, 2020 · 1 min

Another joint university project, AirSense was a hardware and software solution built as part of my coursework for the Internet of Things, Security and Cloud course at the University of Edinburgh.

AirSense is an IoT solution to allow individuals to track their own ex- posure to air pollution, and enable them to make informed changes to improve their personal health.

The system makes use of a Bluetooth Classic (2.1) sensor pack, that can be attached to a bag, which periodically reads air quality data using a variety of sensors. This data is pushed to a companion app running on the user’s phone, which calculates an air quality metric. This data is then uploaded to the AirSense cloud (operating on Google Firebase), to allow a global map of anonymised air quality data to be built.
